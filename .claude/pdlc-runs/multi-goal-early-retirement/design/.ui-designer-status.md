## Design Decisions

1. **No new color system** — extends existing blue/emerald/amber/red with purple/teal for 6-phase support
2. **No Figma** — per Design Lead's 00-design-mode.md assessment (EXTEND-EXISTING, no external refs)
3. **Allocation visualization is CSS-only** (no Chart.js donut) per assumption A25
4. **Phase cards auto-sort by startAge** — no drag-drop per assumption A14
5. **Empty state prominently features "Load Example" CTA** — reduces first-time setup friction per A23
6. **Deficit suggestion is actionable** — "increase SIP by ₹X OR reduce phase Y by Z%" per AC6
